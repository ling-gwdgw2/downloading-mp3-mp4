ฟีเจอร์และระบบใหม่

  • **UI Layout Reorganization**: ย้ายตัวรายงานความคืบหน้า (Progress Bar) และตัวจัดการโฟลเดอร์เซฟไฟล์ (Settings & Logs) ไปไว้ที่ฝั่งขวา (Right Panel) พร้อมลบกล่องเพลงจำลองเพื่อประหยัดพื้นที่และเข้าถึงโฟลเดอร์ได้รวดเร็ว
  • **One-Click Paste & Auto-Analyze**: เพิ่มปุ่มไอคอนกระดาษ "วาง" ถัดจากกล่องอินพุตเพื่อดึงลิงก์จากคลิปบอร์ดมาใส่และวิเคราะห์ให้ทันทีโดยอัตโนมัติในคลิกเดียว
  • **Keyboard Enter Key Support**: รองรับการกดปุ่ม Enter บนแป้นพิมพ์เพื่อเริ่มสั่งวิเคราะห์หรือค้นหาลิงก์ได้โดยทันทีโดยไม่ต้องพึ่งเมาส์คลิกปุ่ม

การปรับปรุงประสิทธิภาพและแก้ไขบั๊ก

  • **PyInstaller Import Bypass**: แก้ไขบั๊ก FrozenImporter ทำให้แอปดึงเครื่องยนต์ yt-dlp รุ่นอัปเกรดภายนอกจาก bin/yt-dlp-update มาทำงานแทนตัวใน EXE จริงหลังรีสตาร์ทสำเร็จ
  • **Version Normalization Check**: ปรับระบบเทียบเวอร์ชัน PyPI กับแอปผ่าน Tuple จำนวนเต็ม ป้องกันปุ่มอัปเดตแสดงซ้ำซ้อนจากรูปแบบวันที่ (เช่น 2026.07.04 เทียบกับ 2026.7.4)
  • **Close Button & Heartbeat Removal**: ปิดปุ่มกากบาท [X] บน Windows Title Bar และ Alt+F4 (ใช้ Win32 API ctypes) บังคับปิดด้วยปุ่มปิดโปรแกรมใน UI เท่านั้นเพื่อความปลอดภัยในการเคลียร์ Cache

ข้อตกลงการใช้งานและคำเตือนทางกฎหมาย

  • เพื่อการใช้งานส่วนตัวเท่านั้น (Personal Use Only): ซอฟต์แวร์นี้สร้างขึ้นเพื่อวัตถุประสงค์ในการศึกษา การค้นคว้าวิจัย และการทำสำเนาสำรองข้อมูลมัลติมีเดียส่วนบุคคล (Personal Backup) เท่านั้น ห้ามนำไปใช้เพื่อแสวงหาผลประโยชน์ทางการค้าหรือธุรกิจโดยเด็ดขาด
  • การเคารพสิทธิ์ในทรัพย์สินทางปัญญา (Copyright Respect): ผู้ใช้งานจะต้องรับผิดชอบต่อกฎหมายลิขสิทธิ์ด้วยตนเอง โปรดดาวน์โหลดเฉพาะวิดีโอหรือไฟล์เสียงที่ได้รับสิทธิ์ตามกฎหมาย หรือได้รับอนุญาตจากเจ้าของลิขสิทธิ์อย่างเป็นทางการเท่านั้น และห้ามนำไฟล์ที่ดาวน์โหลดไปแจกจ่ายสู่สาธารณะโดยไม่ได้รับสิทธิ์
  • ข้อตกลงการใช้บริการแพลตฟอร์ม (Terms of Service): การดาวน์โหลดไฟล์จาก YouTube และ Spotify อาจขัดต่อข้อตกลงการใช้งาน (Terms of Service) ของทางแพลตฟอร์มนั้นๆ ผู้ใช้งานยอมรับและรับทราบถึงข้อกำหนดและความเสี่ยงในส่วนนี้ด้วยตนเอง
  • การปฏิเสธความรับผิดชอบ (No Liability): ผู้พัฒนาโปรแกรมจะไม่รับผิดชอบต่อการละเมิดกฎหมาย การดำเนินคดีทางลิขสิทธิ์ หรือความเสียหายใดๆ ทั้งสิ้นที่เกิดจากการใช้งานโปรแกรมนี้ที่ผิดวัตถุประสงค์ของผู้ใช้ปลายทาง
