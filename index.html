แปลงไฟล์เสียงได้รวดเร็วทันใจ เลือกบิตเรตได้ตามใจตั้งแต่ 128kbps, 192kbps จนถึงเสียงคมชัดระดับพรีเมียม 320kbps ด้วยแกนประมวลผล FFmpeg

แก้ทางบล็อกอัจฉริยะ

ระบบ Browser Cookies Fallback ตรวจหาและดึงคุกกี้สิทธิ์การเข้าชมจากเบราว์เซอร์หลัก (Chrome, Edge, Firefox, Opera) ป้องกันการถูกปฏิเสธดาวน์โหลดจากบอตได้อย่างมั่นใจ

ติดตามสถานะแบบเรียลไทม์

ฟีเจอร์และระบบใหม่ (New Features)

  • **Single-Page UI Consolidation**: ยุบหน้ารายละเอียดการทำงาน มารวมกับหน้าต่างค้นหาหลักให้เสร็จสรรพในหน้าจอเดียว ไหลลื่น ไม่โหลดเปลี่ยนหน้าจอให้กวนใจ
  • **Browser Cookies Fallback Integration**: ค้นหาโฟลเดอร์เก็บคุกกี้เบราว์เซอร์อัตโนมัติบนคอมพิวเตอร์ และทำระบบสลับทำงานเมื่อดาวน์โหลดวิดีโอบางประเภทติดขัด
  • **Phoebe Visual Identity Addition**: นำเข้าและประดับชุดตัวแสดงผลสัญลักษ์ Phoebe (โมเดลการ์ตูน phoebe0/1/2) สร้างอรรถรสภาพสไตล์อนิเมะเรโทร

การปรับปรุงประสิทธิภาพและแก้ไขบั๊ก (Improvements & Bug Fixes)

  • **Windows Shell Registry Integration**: ปรับเปลี่ยนพาธบันทึกไฟล์ดาวน์โหลดเริ่มต้น ให้ตรวจสอบจากที่อยู่โฟลเดอร์เซฟจริงผ่าน Windows System Explorer Registry
  • **Code Cleanup**: ถอดระบบ GUI แบบ Native Python (`gui_app.py`, `create_logo.py`) ออก เพื่อลดขนาดตัวแอปหลังคอมไพล์ลงกว่า 40%
  • **Local Logging System**: เพิ่มตัวเก็บประวัติเออร์เรอร์ลงไฟล์ `app.log` ภายในเครื่อง คลีนและตรวจสอบจุดติดขัดได้ทันที
